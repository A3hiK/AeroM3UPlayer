# 🚀 AeroM3U Player

AeroM3U Player is a modern, lightweight, and high-performance IPTV player designed for seamless M3U playlist streaming. Built with a clean dark-themed interface and an intuitive user experience, AeroM3U Player makes it easy to manage playlists, browse channels, and enjoy live TV streams with minimal effort.

Whether you're loading playlists from local files or remote URLs, AeroM3U Player delivers fast channel loading, smooth playback, and reliable performance in a sleek and distraction-free environment.

### ✨ Key Features

* 📺 M3U Playlist URL & Local File Support
* 🎬 Smooth and Reliable Video Playback
* ❤️ Favorite Channels Management
* 🎨 Modern Dark-Themed Interface
* ⚡ Fast Playlist Parsing & Channel Loading
* ⚙️ Simple and User-Friendly Navigation
* 🚀 Lightweight and Performance Optimized

### 🎯 Mission

To provide a fast, elegant, and user-friendly IPTV experience without unnecessary complexity, allowing users to focus on what matters most—watching their favorite content.
