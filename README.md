**A modern, lightweight, and high-performance IPTV player for seamless M3U playlist streaming.**

Built with a clean dark-themed interface and an intuitive user experience, **AeroM3U Player** makes it easy to manage playlists, browse channels, and enjoy live TV streams with minimal effort.

Whether you're loading playlists from local files or remote URLs, AeroM3U Player delivers fast channel loading, smooth playback, and reliable performance in a sleek and distraction-free environment.

## 📥 Installation

1. Go to the [Releases](../../releases) page.
2. Download the latest `AeroM3UPlayer-Setup.exe`.
3. Run the installer and follow the on-screen steps.
4. Launch AeroM3U Player and load your M3U playlist (URL or local file).

## 🎯 Mission

To provide a fast, elegant, and user-friendly IPTV experience without unnecessary complexity, allowing users to focus on what matters most — watching their favorite content.
